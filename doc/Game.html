    compare takes two array of colors as arguments. and returns differents feedbacks based upon the result. “zero coincidences” if arrays don’t share any value. “color missplaced” if one or more color guesed are in the secret code but not en that position. “color properly placed” if one or more guessed color are in the secret code in that excat position.

    # File game.rb, line 19
def compare(a, b)
  @attemps -= 1
  feedback_zero if a.intersection(b).length.zero?

  if a == b
    same_code_feedback
  else
    a.intersection(b).each do |i|
      if a.index(i) == b.index(i)
        absolute_feedback
      else
        relative_fedback
      end
    end
  end
end

def retry?
  puts 'Retry?
      Y -Yes
      N -No'
  response = gets.chomp
  case response
  when 'y'
    @restart = true
  when 'n'
    restart = false
  else
    puts 'Type y or n please'
    retry?
  end
end
